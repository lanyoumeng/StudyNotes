hadoop安装

sudo chown -R lanmengyou:lanmengyou /usr/local/hadoop

export PATH=$PATH:/usr/local/hadoop/bin

**启动****Hadoop**

```Bash
cd /usr/local/hadoop
./sbin/start-dfs.sh  #start-dfs.sh是个完整的可执行文件，中间没有空格
jps #判断是否成功启动

cd /usr/local/hadoop/sbin
./start-yarn.sh

#启动Hbase
cd /usr/local/hbase
bin/start-hbase.sh

#进入HBase shell命令行模式
bin/hbase shell  

停止HBase运行：
bin/stop-hbase.sh
```
